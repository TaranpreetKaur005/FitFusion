# FitFusion Frontend

## 📌 Description
FitFusion is a stylish AI-based outfit recommendation UI.

## 🚀 Features
- Modern gradient UI
- Outfit card
- Progress tracking
- JSON-based dynamic content

## ▶️ How to Run
1. Open index.html in browser
OR
2. Use Live Server in VS Code
